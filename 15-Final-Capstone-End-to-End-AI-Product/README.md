# Module Notes

Students will propose and build a complete real-world AI application integrating skills from previous modules. The project must include:

A problem statement with measurable outcomes (e.g., customer support automation, smart report generation, intelligent search)

Data ingestion (structured/unstructured)

Application of an LLM using prompt engineering, LangChain, or Hugging Face

Integration with at least one external API or tool

UI or interface demonstration (CLI, chatbot, or web UI)

Deployment and version control through GitHub


Examples:

An AI assistant for internal HR document search

A chatbot that answers customer product questions using uploaded manuals

A multi-agent research and report generator tool
